### Public Endpoints
- No authentication required
- Accessible to everyone
- Used for: registration, login, config, content

### Protected Endpoints
- Require valid JWT token
- User must be active
- Used for: profile management, basic features

### Verified Endpoints
- Require authentication + verification
- Used for: core dating features (swipes, matches)

### Premium Endpoints
- Require authentication + verification + premium
- Used for: advanced features (undo swipe)
